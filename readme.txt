
Default Gameplay Controls:

arrow keys - move and aim
z - jump
x - shoot


Level editor controls

Q and E to switch between different clickmodes
	clickmodes are "Move" "Create" "Edit" 
	and "Connections" (Connections is only accessible when editing the "connect" value of switches)
Left clicking does most actions
Right clicking cancels current actions
When editing a numeric value such as x/y Backspace removes the last numeral and the number keys add their corresponding numeral.
S opens the save menu
L opens the load menu

In order to get a level to appear in the level select it must be manually added to the LevelList file.
